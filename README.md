# Space Invaders Chekcers game by JavaFX
The game of checkers made with Java and JavaFX, space invaders style.

Copy the `ckeckers` directory  under `lib` in  your local `repository`.

To run the game: 

1. mvn pakcage 

2. mvn clean javafx:run
