
# Chekcers Space Invaders JavaFX Game
The game of checkers made with Java and JavaFX, space invaders style.(still developing).

Copy the `ckeckersengine` directory  under `lib` in  your local `repository`.

To run the game: 

1. mvn clean compile 

2. mvn javafx:run

You can download and run inno setup installer :<a href="https://github.com/alegithub72/alegithub72/raw/main/cinvaders-x64bit-4.0.exe" >cinvaders-x64bit-4.0.exe</a>
